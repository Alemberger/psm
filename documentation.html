Provider Screening Module documentation

This site contains documentation for the open source Provider Screening Module.

For more information, please see the PSM GitHub repository, or join our chat server at chat.opentechstrategies.com.

User guide

You can read the manual for PSM users (state agency staff, system administrators, medical providers, and agents serving providers).

It is also available as a PDF file and as an ePub file.

Developer and administrator documentation

This site publishes the documentation for the Java APIs provided by the PSM.

Please also consult the installation manual and architecture design.
